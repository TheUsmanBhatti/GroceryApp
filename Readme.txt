Steps to Run Project:

1. Setup React Native Environment.
2. Clone the Grocery App.
3. Install Node Modules using "npm i",
4. Run Emulator or Attach any physical device (USB Debugging Mode should be ON).
5. Check If device is connected "adb devices".
6. Run App using "npx react-native run-android"
7. If metro is not started while building app then run command "npm start".

If font are not linked then you can use this command "npx react-native-asset".
